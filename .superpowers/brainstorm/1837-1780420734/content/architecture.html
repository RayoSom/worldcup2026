Arquitectura — World Cup 2026 Predictor

3 módulos independientes sobre el repo existente. Cada uno tiene su ruta, puede desactivarse en config.

Estructura de la aplicación
FRONTEND — Next.js 15 (existente + nuevo)
🏆 /torneo
Rediseño visual (C). Motor ELO+Poisson existente.
src/app/[locale]/torneo/ → existente
⚔️ /partido NUEVO
Selector de equipos → simulación individual → resultados + narrativa.
src/app/[locale]/partido/ → nuevo
🤖 /predictor NUEVO 🔒 CÓDIGO
Chat IA interactivo. Requiere código de acceso. Análisis táctico profundo.
src/app/[locale]/predictor/ → nuevo
BACKEND — API Routes + Config
⚙️ src/config/modules.ts
Un objeto con enabled: true/false por módulo. Para desactivar: cambiar a false y rebuild.
🔌 /api/ai-chat
Valida código → llama Claude API → stream de respuesta. Vercel AI SDK.
📦 src/data/team-narratives.json
48 equipos × análisis pre-generado con Claude. Se genera una vez offline. Costo: ~$0.10 total.
